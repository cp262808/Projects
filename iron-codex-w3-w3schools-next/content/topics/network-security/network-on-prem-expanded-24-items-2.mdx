<summary>Network → Zero Trust Segmentation</summary>
<p class="tags"><b>Tags:</b> Microsegmentation, ZTA</p>
<p><b>Question:</b> Are HVA zones isolated with default-deny microsegmentation and east-west controls?</p>
<p><b>Applicable Requirements:</b><br/>- NIST 800-53: SC-7, AC-4<br/>- ISO 27001: A.13.1.3</p>
<p><b>Applicability:</b> Datacenter and hybrid networks</p>
<p><b>Expected Result:</b> Identity/workload-based policy; deny lateral; egress pinned.</p>
<p><b>Why It Matters:</b> Reduces ransomware spread and escalation.</p>
<p><b>Technical Breakdown:</b></p>
<ol>
<li>Identity-aware proxies; SDP; combine host firewall + network ACLs.</li>
</ol>
