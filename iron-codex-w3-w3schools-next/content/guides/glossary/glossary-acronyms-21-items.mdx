<div class="card"><h1 style="margin:0">Glossary (Acronyms)21 items</h1></div>
<details class="section" id="glossary-acronyms"><summary>Glossary (Acronyms)<span class="badge">21 items</span></summary><details><summary>API</summary><p>Application Programming Interface — a contract for software-to-software interaction.</p><ul><li>See also: REST, GraphQL, gRPC.</li></ul></details><details><summary>SSO</summary><p>Single Sign-On — one identity session to access many apps.</p><ul><li>Often implemented with SAML or OpenID Connect (OIDC).</li></ul></details><details><summary>OAuth 2.0</summary><p>Authorization framework for delegated access; not an identity protocol.</p><ul><li><a href="https://www.rfc-editor.org/rfc/rfc6749" target="_blank">RFC 6749</a></li></ul></details><details><summary>OIDC</summary><p>OpenID Connect — identity layer on top of OAuth2 that provides user authentication.</p><ul><li><a href="https://openid.net/specs/openid-connect-core-1_0.html" target="_blank">OpenID Connect Core</a></li></ul></details><details><summary>JWT</summary><p>JSON Web Token — a compact, signed token format for conveying claims.</p><ul><li><a href="https://www.rfc-editor.org/rfc/rfc7519" target="_blank">RFC 7519</a></li></ul></details><details><summary>mTLS</summary><p>Mutual TLS — both client and server present certificates.</p><ul><li>Used for service-to-service identity in zero trust architectures.</li></ul></details><details><summary>BYOK</summary><p>Bring Your Own Key — vendor uses your key in their KMS.</p></details><details><summary>HYOK</summary><p>Hold Your Own Key — your keys never leave your HSM boundary.</p></details><details><summary>KMS</summary><p>Key Management Service — managed cryptographic key lifecycle and usage.</p></details><details><summary>HSM</summary><p>Hardware Security Module — tamper-resistant cryptographic hardware.</p></details><details><summary>ZTNA</summary><p>Zero Trust Network Access — per-app access with strong identity and device posture.</p><ul><li><a href="https://csrc.nist.gov/publications/detail/sp/800-207/final" target="_blank">NIST SP 800-207</a></li></ul></details><details><summary>PAM</summary><p>Privileged Access Management — broker and monitor privileged sessions.</p></details><details><summary>PIM</summary><p>Privileged Identity Management — JIT elevation for roles.</p></details><details><summary>SSPM</summary><p>SaaS Security Posture Management — continuous config monitoring of SaaS apps.</p></details><details><summary>CASB</summary><p>Cloud Access Security Broker — visibility and control for SaaS usage.</p></details><details><summary>SIEM</summary><p>Security Information &amp; Event Management — log aggregation, detection, and response.</p></details><details><summary>SOAR</summary><p>Security Orchestration, Automation, and Response — automated enrichment and actions.</p></details><details><summary>WAF</summary><p>Web Application Firewall — blocks common web attacks at the edge or app.</p></details><details><summary>DLP</summary><p>Data Loss Prevention — monitor/block data exfiltration.</p></details><details><summary>SBOM</summary><p>Software Bill of Materials — inventory of software components.</p></details><details><summary>VEX</summary><p>Vulnerability Exploitability eXchange — signals exploitability for vulnerabilities.</p></details></details>
