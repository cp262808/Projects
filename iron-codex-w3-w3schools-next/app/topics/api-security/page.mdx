# API Security <Badge>beta</Badge>

<Callout type="info" title="Why this matters">
APIs expose your core data plane. Harden authn/z and validate inputs at every hop.
</Callout>

## Quick start

<Steps>
  <Step>Enable **JWT** validation at the edge and enforce audience/issuer checks.</Step>
  <Step>Apply **rate limits** per client and method.</Step>
  <Step>Require **mTLS** between services for east‑west traffic.</Step>
</Steps>

## Sample: verify JWT (Node)

```ts
import jwt from 'jsonwebtoken'
import { Request, Response, NextFunction } from 'express'

export function verifyJwt(req: Request, res: Response, next: NextFunction) {
  const hdr = req.headers.authorization || ''
  const token = hdr.replace(/^Bearer\s+/i, '')
  try {
    const decoded = jwt.verify(token, process.env.JWT_PUBLIC_KEY!, { algorithms: ['RS256'] })
    ;(req as any).user = decoded
    return next()
  } catch (err) {
    return res.status(401).json({ error: 'Invalid token' })
  }
}
```
